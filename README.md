# bash-data-observability-sre-systems-lab
This repository is a production-inspired observability and automation lab built entirely with Bash scripting.
It demonstrates how to design monitoring systems, log pipelines, and self-healing infrastructure without relying on heavy frameworks—focusing instead on core Linux capabilities and engineering fundamentals.
It is true, however, in production, we must manage data from several sources, in these conditions frameworks like Splunk, Dynatrace, ELK, Cribl, etc., are essential. So we will also talk about frameworks.

This repository covers my areas of interest:

- SRE (SLI/SLO/SLA, Error Budgets, Reliability, Incident Response)
